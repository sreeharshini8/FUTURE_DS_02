# Telecom Customer Retention & Churn Dashboard

## 📌 Project Overview

This project focuses on analyzing customer churn in the telecom industry using Power BI.

The dashboard provides insights into customer retention patterns, churn behavior, risk segmentation, and factors influencing customer loss.

The main objective of this project is to identify the key reasons behind customer churn and help businesses improve customer retention strategies.

---

# 📊 Dashboard Features

* Total Customers KPI
* Churned Customers KPI
* Churn Rate Analysis
* Average Monthly Charges
* Average Customer Lifetime
* Tech Support Impact on Churn
* Customer Churn by Tenure Group
* Contract Type vs Churn
* Customer Risk Segmentation
* Customer Churn Distribution
* Interactive Filters & Slicers

---

# 🛠️ Tools & Technologies Used

* Power BI
* DAX (Data Analysis Expressions)
* Data Cleaning & Transformation
* Data Visualization
* Business Intelligence Concepts

---

# 📂 Dataset Information

The dataset contains telecom customer information including:

* Customer ID
* Contract Type
* Tenure
* Monthly Charges
* Tech Support
* Payment Method
* Customer Churn Status
* Risk Segmentation

---

# 📈 Key Insights

* Around 25–30% of customers are churning.
* Month-to-month customers show the highest churn rate.
* New customers (0–12 months) are more likely to leave.
* Customers without tech support churn more frequently.
* High monthly charges increase customer churn risk.
* Long-term contracts improve customer retention significantly.

---

# 🎯 Business Impact

This dashboard helps businesses:

* Identify high-risk customers
* Improve retention strategies
* Understand churn behavior
* Enhance customer satisfaction
* Reduce revenue loss caused by churn
